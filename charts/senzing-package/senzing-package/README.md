# senzing-package

A job to install 
[Senzing_API.tgz](https://github.com/Senzing/knowledge-base/blob/master/WHATIS/senzing-api-tgz.md)
package onto `/opt/senzing`.
Normally, `/opt/senzing` is a Persistent Volume.
